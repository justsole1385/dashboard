# CapacitaciónJS

## Descripción general

Esta carpeta contiene la evidencia de capacitación en JavaScript. 

## Unidades revisadas

### Unidad 2: Strings I

Esta unidad introduce el manejo de cadenas de texto en JavaScript. 
Se estudia cómo crear strings con comillas simples o dobles, obtener su longitud con `.length`, convertir texto a mayúsculas o minúsculas con `.toUpperCase()` y `.toLowerCase()`, acceder a caracteres mediante índices y utilizar métodos como `.at()` y `.substring()`. También se revisa la concatenación con `+` y el uso de template strings para interpolar variables dentro de textos.

### Unidad 3: Numbers

Esta unidad aborda operaciones y conversiones numéricas en JavaScript. 
Se revisa cómo convertir números a texto con `.toString()`, convertir texto a número con `Number.parseInt(valor, 10)`, interpretar el valor `NaN`, usar el operador de residuo `%` y aplicar funciones matemáticas como `Math.round()`, `Math.floor()` y `Math.ceil()`. 

### Unidad 4: Variables

Esta unidad explica la declaración de variables con `let` y `const`. 
Se diferencia el uso de `let` para valores que pueden reasignarse y `const` para valores que no deben reasignarse. También se menciona que `var` pertenece a versiones antiguas de JavaScript y que las variables declaradas con `let` y `const` tienen alcance de bloque. 

### Unidad 5: Condiciones

Esta unidad presenta el uso de estructuras condicionales en JavaScript. 
Se revisa cómo utilizar `if` para ejecutar código cuando una condición se cumple y `else` para manejar casos alternativos . También se explica que algunas condiciones cortas pueden escribirse con el operador ternario, aunque se recomienda priorizar la legibilidad del código.

## Relación con el proyecto

Los contenidos de esta capacitación aportan la base de JavaScript necesaria para construir un dashboard interactivo. 
El manejo de strings permite trabajar con categorías como género o plataforma; el manejo de números permite procesar variables como horas de sueño o niveles de estrés; las variables permiten almacenar resultados y configuraciones; y las condiciones permiten controlar filtros, validaciones y comportamientos dinámicos de la interfaz.
