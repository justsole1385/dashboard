# Dashboard Interactivo: Impacto de las Redes Sociales en la Salud Mental Adolescente

## Objetivo del dashboard

El objetivo de este dashboard es analizar de forma visual e interactiva la relación entre el uso de redes sociales y distintos indicadores asociados al bienestar de adolescentes, como horas de sueño, tiempo de pantalla antes de dormir, rendimiento académico, actividad física, estrés, ansiedad, nivel de adicción y etiqueta de depresión.

El dashboard permitirá transformar los datos del dataset en información útil para identificar patrones, comparar grupos y apoyar la toma de decisiones preventivas en contextos educativos. A través de indicadores, gráficos y filtros, el usuario podrá observar cómo varían los hábitos digitales y los indicadores de salud mental según edad, género, plataforma utilizada y nivel de interacción social.

## Dataset seleccionado
- **Nombre:** Social Media Impact on Teen Mental Health
- **URL:** https://www.kaggle.com/datasets/algozee/teenager-menthal-healy

## Propósito general

Desarrollar un dashboard interactivo que facilite la comprensión de los datos relacionados con el uso de redes sociales en adolescentes, permitiendo visualizar indicadores clave, comparaciones entre variables y posibles patrones de riesgo asociados al bienestar mental.

## Usuario objetivo

El dashboard está dirigido principalmente a orientadores educativos, psicólogos escolares, docentes tutores y directivos de instituciones educativas que necesiten interpretar información sobre hábitos digitales, sueño, rendimiento académico, actividad física e indicadores de estrés o ansiedad en adolescentes.
